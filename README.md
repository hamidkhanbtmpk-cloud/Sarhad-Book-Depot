# 📚 Sarhad Book Depot - Inventory & E-commerce System

A comprehensive management system designed for **Sarhad Book Depot** to streamline inventory tracking and online sales operations. Built with a focus on efficiency and ease of use for local businesses.

## 🚀 Features
* **Inventory Tracking:** Real-time monitoring of book stocks, categories, and pricing.
* **E-commerce Ready:** Integrated backend for managing online orders and customer inquiries.
* **User Dashboard:** Secure and intuitive interface for administrators.
* **Database Management:** Optimized data storage for thousands of book titles.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite / PostgreSQL (depending on setup)
* **Styling:** Responsive Design for mobile and desktop views
